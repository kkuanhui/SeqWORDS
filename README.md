# SeqWORDS
This is Python SeqWORDS package. SeqWORDS is an unsupervised Chinese word segmentation method, which not demand a dictionary.
### Installation
```bash
pip install SeqWORDS
```
### Usage
```python
import SeqWORDS

for i in range(100):
    print("Fuck YOU!)
```
